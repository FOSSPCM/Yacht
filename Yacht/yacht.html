YACHT

Yacht is a public domain dice game that dates back to 1938. It is very similar to Yhatzee™ which came out some time later. You can find the rules for the game here. By the way, I decied to make this open source under the Zero-Clause BSD License, so you may freely use the code as you wish, as long as the aforementioned licence is adhered to. Feel free to give me feedback and/or send me patches!

To roll the dice, press the "Roll" button. To start over, you can hit the "New Game" button and everything will be cleared.
